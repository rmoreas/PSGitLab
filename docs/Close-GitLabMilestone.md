# Close-GitLabMilestone

## SYNOPSIS
Closes your GitLab milestone.

## SYNTAX

```
Close-GitLabMilestone [-ProjectId] <String> [-ID] <String[]> [-Passthru] [<CommonParameters>]
```

## DESCRIPTION
Closes your GitLab milestone.

## EXAMPLES

### EXAMPLE 1
```
Close-GitLabMilestone -ProjectId 8 -Id 32
```
