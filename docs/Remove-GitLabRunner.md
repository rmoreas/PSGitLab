# Remove-GitLabRunner

## SYNOPSIS
Removes a runner.

## SYNTAX

```
Remove-GitLabRunner [-Id] <Int32> [<CommonParameters>]
```

## DESCRIPTION
Removes a runner from GitLab.

## EXAMPLES

### EXAMPLE 1
```
Remove-GitLabRunner -Id 400
```

## PARAMETERS

### -Id
