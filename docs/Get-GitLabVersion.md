# Get-GitLabVersion

## SYNOPSIS
Retrieves version information for this GitLab instance.

## SYNTAX

```
Get-GitLabVersion [<CommonParameters>]
```

## DESCRIPTION
Retrieves version information for this GitLab instance.

## EXAMPLES

### EXAMPLE 1
```
Get-GitLabVersion
```
