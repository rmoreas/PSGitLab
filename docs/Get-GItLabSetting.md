# Get-GitLabSetting

## SYNOPSIS
Get all of the GitLab instance global settings. 

## SYNTAX

```
Get-GitLabSetting [<CommonParameters>]
```

## DESCRIPTION
Get all of the GitLab instance global settings. 
Uses the appliation API endpoint. 

## EXAMPLES

### EXAMPLE 1
```
Get-GitLabSetting
```
