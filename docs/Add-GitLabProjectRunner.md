# Add-GitLabProjectRunner

## SYNOPSIS
Enable an available specific runner in the project.

## SYNTAX

```
Add-GitLabProjectRunner [-RunnerId] <Int32> [-ProjectId] <Int32> [<CommonParameters>]
```

## DESCRIPTION
Enable an available specific runner in the project.

## EXAMPLES

### EXAMPLE 1
```
Add-GitLabProjectRunner -RunnerId 400 -ProjectId 500
```
